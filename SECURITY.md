## Reporting Security Issues

We take all security reports seriously. If you discover a potential security
issue in this project, please **do not** create a public GitHub issue.

Instead, report it to AWS Security via
[https://aws.amazon.com/security/vulnerability-reporting/](https://aws.amazon.com/security/vulnerability-reporting/)
or email [[email protected]](mailto:[email protected]). You may also use the
repository's private [GitHub Security Advisories](../../security/advisories)
feature.

Please do **not** create a public issue for security problems.

Thank you for helping keep this project and its users safe.
